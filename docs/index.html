Spielsysteme

Alles in einem Bot.

Kein externes Interface, kein Account-System. Alles per Slash Command direkt im Discord-Server.

⏱️
Tick-System
Alle 6h verarbeitet der Bot alle Firmen: Salary, Events, NPC-Wachstum, Projekt-Fortschritt.
Core Loop
🤖
NPC-Konkurrenten
Firmen mit 3 Persönlichkeiten wachsen automatisch. Aggressive greifen aktiv an.
Lebendige Welt
🔬
Tech-Tree
15 Nodes in 3 Säulen: Operations, R&D, People. Research Points passiv pro Tick.
Progression
🗡️
Sabotage
4 Aktionen: Leak, Hiring War, Fake PR, Infra-Angriff — mit Backfire-Mechanik.
PvP
📋
30 Projekte
6 pro Markt. Risiko-Roll mit 4 Outcomes. Schalten Märkte und Nodes frei.
Content
🌟
Prestige-System
Hard Reset (+10% Legacy-Mult.) oder Soft Prestige (IPO, Markt-Dominanz).

So geht's

In 4 Schritten loslegen

01
Bot einladen
Den Bot über den Invite-Link zum Discord-Server hinzufügen.
02
/start eingeben
Firmenname, Markt und Startertyp per Discord-Buttons wählen.
03
Team aufbauen
Mit /hire ein Team aufbauen, erstes Projekt starten, KPIs im Blick behalten.
04
Dominieren
NPC-Firmen übernehmen, Tech-Tree erforschen, Konkurrenten sabotieren.
Slash Commands

Alles per Command.

CommandWas es macht
/startFirma gründen — Markt und Startertyp per Discord Buttons
/reportCompany Dashboard mit allen KPIs, Trends und Valuation
/hire [rolle]Mitarbeiter einstellen — zufälliger Name, gewürfelte Stats
/fire [name]Mitarbeiter feuern — Morale sinkt im Team
/project [aktion]list / start / status / boost / cancel
/research [aktion]tree / pick / status — Tech-Tree verwalten
/acquire [firma]NPC-Firma übernehmen (ab 3 Mitarbeitern)
/fundraiseVC-Pitch starten (ab Tick 5)
/expand [markt]Neuen Marktbereich betreten
/sabotage @userleak / hiring_war / fake_pr / infra (ab $50k Valuation)
/prestigeHard Reset oder Soft Prestige (ab $1M Valuation)
/marketServer-Leaderboard aller Firmen nach Valuation
/profile [firma]Öffentliches Firmenprofil anzeigen
/legacyPrestige-History und alle Achievements
/logLetzte Events der eigenen Firma
